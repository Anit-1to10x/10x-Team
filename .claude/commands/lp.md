# /lp

Shortcut for `/landing-page` command. Developed by Team 10x.in.

## Usage

```
/lp [action] [project-name]
```

## Actions

- `/lp new` - Start a new landing page project
- `/lp resume <name>` - Continue an existing project
- `/lp list` - List all projects

## Example

```bash
/lp new
/lp resume my-saas-landing
/lp list
```
